Instruction for use

Congratulations for creating your webfont bundle!

With this package you got everything you need to start using SuitcaseType Foundry's webfonts. There are just two simple steps to get going.

All you need is just copy all webfont (eot, woff, svg) files to your project resource folder. Assume its location is /resources.

Then add this css snippet to your css file: 

@font-face {
  font-family: 'Purista_Light';
  src: url('/resources/Purista_Light.eot?#iefix') format('embedded-opentype'),
       url('/resources/Purista_Light.woff') format('woff'),
       url('/resources/Purista_Light.svg') format('svg'),
       local('☺');
;
}

From this point you can use it as every other font family in your css files like this:

#content h1 {
  font-family: 'Purista_Light';
  font-weight: normal;
  font-style: normal;
}

Remember to assign the weight and style of the font, especially if you don't have the entire family.

Or just add our supplied file to your html head: 

<link href='Purista_Light.css' rel='stylesheet' type='text/css' />

Also look at the source of this file for the further examples of the advanced use.
